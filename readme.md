# Srivatsa Kamballa Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects, and experiences as a CS Grad Student and web developer.

## Features

- Responsive design for various screen sizes
- Interactive UI elements
- Sections for About Me, Skills, and Projects
- Dark/Light mode toggle
- Sound toggle
- Social media links
- Resume download option

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll) library

## Setup

1. Clone the repository:
git clone https://github.com/YourUsername/srivatsa-kamballa-portfolio.git

2. Navigate to the project directory:
cd srivatsa-kamballa-portfolio

3. Open `index.html` in your preferred web browser.

## Usage

Navigate through the different sections using the menu. Toggle between dark and light modes, and enable/disable sound using the settings button on the homepage.
