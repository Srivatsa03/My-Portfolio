fixes that left my machine

Open source

Small, careful patches to tools other people depend on. I care less about the line count and more about catching the second broken file nobody noticed.

  • deepset-ai/haystack 20k+ stars PR #11670 · merged

    Silenced noisy logs on empty inputs

    HTMLToDocument was dumping ERROR and WARNING lines whenever it got an empty ByteStream, because trafilatura and lxml log their parse failures internally instead of raising, so the existing try/except never caught them. I skipped empty streams before extraction, added a test and a release note, and cleared full CI. A maintainer reviewed and merged it.

  • BerriAI/litellm 48k+ stars PR #30764 · open, CI green

    Stopped short secrets leaking into logs

    While reading the code I noticed SensitiveDataMasker returned any secret of eight characters or fewer verbatim, an off-by-one threshold plus a zero-length boundary mask, so short redis passwords, API keys, and tokens were landing in logs and the admin UI. I made short values mask by default, kept two legitimate non-secret callers working through dependency injection, added regression tests, and untangled the CI failures that cascaded through shared callers.

  • BerriAI/litellm 48k+ stars PR #29693 · open, review 5/5

    Fixed a 10x cost-tracking error

    amazon.titan-embed-text-v2 was priced ten times too high in the cost maps, quietly inflating everyone's budget reports. I checked the number against AWS Bedrock pricing, fixed the primary and backup maps in sync so they would not drift apart again, and added a regression test with isolated monkeypatching.
